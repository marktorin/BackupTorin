# BackupTorin

Простой бот python для локального резервного копирования файлов.

**Возможности:**
- Использование своего api telegram
- Резервное копирование по расписанию
- Принудительное резервное копирование
- Отправка уведомлений в телеграм
- Выбор компрессии (zip, tar.gz, tar.bz2)
- Полное логирование
